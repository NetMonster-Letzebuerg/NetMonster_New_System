Guia Completo para Mapear PDOs com o JOSM

Este guia explica como mapear PDOs (Pontos de Distribuição Óptica) usando o editor JOSM.

Passos Detalhados para Mapeamento de PDOs usando o JOSM:

1. Baixar e Instalar o JOSM

O JOSM é um editor de OpenStreetMap para computadores. Você pode baixá-lo pelo link: JOSM Download.

2. Configurar Imagens de Satélite no JOSM

Após instalar o JOSM, é importante configurar o uso de imagens de satélite para ajudar na localização precisa dos PDOs.

  • No menu superior, vá em Imagens de Fundo e selecione Bing. Isso permitirá que você visualize as imagens de satélite da Bing como fundo do mapa.
  • Se necessário, você também pode selecionar outras opções disponíveis, como Mapbox ou Esri, dependendo da clareza das imagens na sua área de interesse.

3. Baixar Área de Informações no JOSM

Antes de começar a editar, você precisa baixar os dados da área em que vai trabalhar.

  • Clique no ícone de seta verde para baixo na barra de ferramentas (ou vá em Ficheiro > Descarregar Dados).
  • Na janela que se abrir, selecione a área que deseja editar usando a ferramenta de seleção e clique em Baixar.
  • Isso carregará todos os dados existentes da área selecionada, permitindo que você edite e adicione informações.

4. Abrir e Navegar pelas Imagens de Satélite

Com as imagens de satélite configuradas, você pode navegar até a área onde o PDO está localizado.

  • Use a ferramenta de zoom e pan para aproximar a área desejada e encontrar a localização exata do PDO.

5. Criar um Nó (Node) no JOSM

Após identificar a localização do PDO, clique no ícone de Criar Nó (atalho A).

  • Posicione o cursor na localização desejada no mapa e clique para criar o nó.

6. Adicionar Tags ao Nó

Selecione o nó criado e adicione as seguintes tags para descrever o PDO corretamente.

  • Para PDO em Parede:
    • operator=DIGI
support=wall_mounted
telecom:medium=fibre
telecom=distribution_point
ref=XPTO
  • Para PDO em Poste:
    • operator=DIGI
support=pole
telecom:medium=fibre
telecom=distribution_point
ref=XPTO
  • Nota: Use a referência do PDO (ref) se estiver visível. Caso contrário, insira XPTO ou XXXX.

7. Salvar e Enviar as Alterações

Após adicionar as tags necessárias, salve suas mudanças e envie-as para o OpenStreetMap.

  • No menu superior, clique em Ficheiro > Carregar Alterações para enviar suas edições ao banco de dados do OpenStreetMap.

Dicas Adicionais:

  • Ajuste de Precisão: Utilize o máximo de zoom possível para garantir a precisão ao posicionar o nó do PDO.
  • Verificação de Dados: Antes de enviar suas alterações, revise as informações adicionadas para evitar erros ou omissões.
  • Colaboração: Participe da comunidade OpenStreetMap para aprender e compartilhar experiências com outros mapeadores.
