1. Aceitação dos Termos

Ao acessar ou utilizar este serviço, você concorda com os Termos de Serviço descritos abaixo. Se você não concordar com estes termos, por favor, não utilize o serviço.

2. Coleta e Processamento de Dados de Antenas

2.1. Coleta de Dados de Antenas: Este serviço coleta informações de antenas em Portugal para fornecer dados sobre a cobertura de rede e qualidade do sinal.

2.2. Processamento e Compartilhamento: As informações coletadas são processadas e enviadas para uma base de dados pública, acessível a todos os usuários. Nenhuma informação pessoal identificável é armazenada ou compartilhada.

3. Uso Adequado

Você concorda em utilizar este serviço de maneira adequada e em conformidade com todas as leis e regulamentações aplicáveis em Portugal. O serviço destina-se a fornecer informações de antenas de maneira pública e colaborativa.

4. Privacidade e Dados Pessoais

4.1. Privacidade: Comprometemo-nos a proteger a privacidade dos usuários. Nenhuma informação pessoal identificável é armazenada ou compartilhada na base de dados pública.

4.2. Dados Anônimos: As informações enviadas para a base de dados pública são anonimizadas e não rastreiam usuários individualmente.

4.3. Nickname e Email (Opcionais): O fornecimento de um nickname e endereço de e-mail é voluntário e usado apenas para facilitar o envio de dados.

4.4. Uso do Nickname para Estatísticas: O nickname fornecido pode ser utilizado para a formação de estatísticas no site, mantendo a privacidade dos usuários.

4.5. Localização dos Dados: Todos os dados são hospedados no território da União Europeia, em conformidade com o Regulamento Geral de Proteção de Dados (RGPD) da União Europeia.

5. Propriedade Intelectual

O conteúdo fornecido através do serviço é protegido por direitos autorais, marcas registradas e outros direitos de propriedade intelectual. Você concorda em respeitar esses direitos.

6. Código e Base de Dados Open-Source

6.1. O código-fonte deste site e a base de dados são open-source, disponíveis publicamente para consulta, contribuição e uso pela comunidade.

6.2. Você pode acessar o repositório do projeto no GitHub: NetMonster Portugal no GitHub.

7. Alterações nos Termos de Serviço

Reservamo-nos o direito de modificar estes Termos de Serviço a qualquer momento. As alterações entrarão em vigor imediatamente após a publicação. O uso contínuo do serviço após tais alterações constituirá sua aceitação dos Termos modificados.
