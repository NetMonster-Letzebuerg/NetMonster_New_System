Sobre Nós

Estatísticas e dados em tempo real das redes móveis de Portugal.

O que é o NetMonster Portugal?

O NetMonster Portugal é um projeto colaborativo criado para mapear todas as antenas de telecomunicações e PDO's (Pontos de Desembarque Óptico) em Portugal. Nosso objetivo é fornecer informações precisas e atualizadas sobre a cobertura de rede, qualidade do sinal e localização das infraestruturas em todo o país.

Como Surgiu?

Este projeto nasceu da comunidade de entusiastas de telecomunicações no grupo do Telegram CellMapper Portugal. A partir das discussões e colaborações no grupo, percebemos a necessidade de centralizar e compartilhar dados sobre as antenas e PDO's de forma acessível e útil para todos.

Como Mapear Antenas?

Para mapear antenas, você precisa usar o aplicativo NetMonster. Ele permite coletar dados sobre as antenas e enviá-los para a nossa plataforma. Para saber como configurar e usar o aplicativo, visite a nossa página de Guias.

Como Mapear PDO's?

Para mapear PDO's, utilizamos dados do OpenStreetMap. Você pode contribuir diretamente no OpenStreetMap, adicionando ou atualizando informações sobre PDO's. Esses dados são então utilizados na nossa plataforma através da API do OpenStreetMap, permitindo que todos os PDO's sejam exibidos em um único mapa.

Como Posso Contribuir?

Você pode contribuir de duas maneiras:

  • Mapear Antenas: Use o aplicativo NetMonster para coletar e enviar dados de antenas. Para instruções detalhadas, acesse a página de Guias.
  • Mapear PDO's: Contribua diretamente no OpenStreetMap, adicionando ou atualizando informações sobre PDO's. Esses dados serão automaticamente integrados ao nosso mapa.

Junte-se à Comunidade

Participe do nosso grupo no Telegram CellMapper Portugal para discutir, colaborar e acompanhar as últimas atualizações do projeto. Sua contribuição é fundamental para o sucesso do NetMonster Portugal!
